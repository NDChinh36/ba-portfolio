## 1. Test Scope

### 1.1 Phạm vi UAT

UAT này kiểm thử toàn bộ tính năng **Futures Position Calculator**, bao gồm:

| Nhóm tính năng | Phạm vi |
|---|---|
| **PnL Calculator** | Tính PnL, ROI, phí giao dịch cho Long / Short |
| **Target Price Calculator** | Tính giá mục tiêu từ PnL mong muốn (tuyệt đối hoặc %) |
| **Liquidation Price Calculator** | Tính giá thanh lý theo 2 mode: có / không có số lượng |
| **Tab Switch & Default Params** | Dữ liệu mặc định được giữ nguyên khi chuyển tab |
| **Copy & Fill** | Điền thông số từ Calculator lên Order Form |
| **Fee Token Handling** | Xử lý logic phí khi dùng Native Token / Stablecoin |
| **Size Validation** | Kiểm tra ràng buộc số lượng và khối lượng lệnh |
| **Tutorial & Onboarding** | Intro tooltip và tutorial 5 bước lần đầu sử dụng |

**Ngoài phạm vi:**
- Luồng đặt lệnh sau khi Copy & Fill (thuộc Order Form UAT)
- Quản lý vị thế, margin, funding rate
- API / Backend performance testing
